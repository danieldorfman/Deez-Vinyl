DEEZ VINYL X — ממשק חדש

מה השתנה:
- החתול הוסר לחלוטין.
- Shuffle ו-Repeat הוחלפו באייקוני SVG נקיים שמתאימים לעיצוב.
- ממשק חדש בסגנון זכוכית כהה וניאומורפי.
- כפתור Play מרכזי עם Glow עדין.
- כפתורי Previous/Next חדשים.
- סליידרים, טיפוגרפיה, מרווחים וצללים עוצבו מחדש.
- כל פונקציות Spotify נשמרו.

הפעלה:
npm install
npm run check
npm start
